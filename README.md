# bhargavchippada.github.io

Personal website built with Next.js 15, TypeScript, and Tailwind CSS v4.

## Tech Stack

- **Framework**: Next.js 15 (App Router, static export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Theme**: Dark/light toggle via next-themes
- **Deploy**: GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npx serve out
```

Deployment is automatic on push to `master` via GitHub Actions.
